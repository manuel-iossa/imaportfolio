Descrizione

La Bottega Dello Sport

Software gestionale in C# Windows Form

Descrizione

Questo progetto, sviluppato in quarto insieme al mio compagno di classe Scortechini Nicolò, consisteva nel creare un software gestionale in Windows Form.
Per realizzare questo software abbiamo messo insieme la cosa che ci accumuna, il calcio, ed abbiamo deciso di creare un software gestionale per un rivenditore di maglie da calcio.
